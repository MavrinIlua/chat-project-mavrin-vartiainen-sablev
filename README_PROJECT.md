# ChatApp — Веб-чат в реальном времени

> Простой и удобный чат прямо в браузере — без установки приложений.

---

## Что это такое?

**ChatApp** — учебный веб-чат, где несколько пользователей могут переписываться в реальном времени.
Достаточно открыть браузер, придумать никнейм и пароль — и ты уже в чате.

---

## Возможности

- **Умное приветствие** — "Доброе утро", "Добрый день", "Добрый вечер" или "Доброй ночи" в зависимости от времени
- **Вход по никнейму и паролю** — первый раз вводишь данные и регистрируешься автоматически
- **История сообщений** — переписка сохраняется в базе данных
- **Время и автор** — у каждого сообщения видно кто и когда написал
- **Группировка по дням** — дата показывается один раз при переходе на новый день
- **Защита от чужого входа** — чужой никнейм без пароля не займёшь

## Как запустить проект

1. Открыть папку проекта в VS Code
2. Открыть терминал: меню Terminal → New Terminal
3. В терминале должно быть написано `(venv)` в начале строки — это виртуальное окружение
4. Написать команду и нажать Enter:
   ```
   python app.py
   ```
5. Открыть браузер и перейти: `http://127.0.0.1:5000`
6. Остановить сервер: нажать `Ctrl+C` в терминале

## Версии проекта

### SQLite-версия (`sqlite_version/`)
- Использует файловую базу данных SQLite (`chat.db`)
- Поля пользователя: `nickname`, `password`
- Чат: групповой (все видят все сообщения)
- Файлы: `app_backup.py`, `static/`, `templates/`

### MySQL-версия (`mysql_version/`)
- Использует серверную базу данных MySQL
- Поля пользователя: `name`, `surname`, `login`, `password`
- Чат: 1-на-1 (выбор собеседника из списка)
- Файлы: `app.py`, `static/`, `templates/`

---

## Какой файл использовать?

- **Для SQLite**: используйте файлы из папки `sqlite_version/` или корневой `app_backup.py`
- **Для MySQL**: используйте файлы из папки `mysql_version/`
