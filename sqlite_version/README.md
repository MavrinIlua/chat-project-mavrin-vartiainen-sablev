# SQLite-версия ChatApp

> Простой и удобный чат прямо в браузере — без установки приложений. Версия на SQLite.

---

## Что это такое?

**ChatApp SQLite** — учебный веб-чат, где несколько пользователей могут переписываться в реальном времени.

---

## Возможности

- **Умное приветствие** — "Доброе утро", "Добрый день", "Добрый вечер" или "Доброй ночи" в зависимости от времени
- **Вход по никнейму и паролю** — первый раз вводишь данные и регистрируешься автоматически
- **История сообщений** — переписка сохраняется в базе данных SQLite
- **Время и автор** — у каждого сообщения видно кто и когда написал
- **Группировка по дням** — дата показывается один раз при переходе на новый день
- **Защита от чужого входа** — чужой никнейм без пароля не займёшь
- **Динамическое обновление** — сообщения появляются без перезагрузки страницы

## Технологии

| Что | Зачем |
|-----|-------|
| Python 3.12 + Flask | Сервер, маршруты, логика |
| SQLite | База данных (пользователи и сообщения) |
| HTML + Jinja2 | Страницы сайта |
| CSS | Внешний вид |
| JavaScript (fetch) | Динамика без перезагрузки |
| Werkzeug | Хеширование паролей |

## База данных SQLite

### Таблица users (пользователи)

| Поле | Тип | Описание |
|------|-----|----------|
| id | INTEGER (PK, AUTOINCREMENT) | Уникальный ID пользователя |
| nickname | TEXT (UNIQUE, NOT NULL) | Никнейм |
| password | TEXT (NOT NULL) | Пароль (хешированный) |

### Таблица messages (сообщения)

| Поле | Тип | Описание |
|------|-----|----------|
| id | INTEGER (PK, AUTOINCREMENT) | Уникальный ID сообщения |
| nickname | TEXT (NOT NULL) | Кто написал |
| text | TEXT (NOT NULL) | Текст сообщения |
| timestamp | TEXT (NOT NULL) | Дата и время в формате "DD.MM.YYYY|HH:MM" |

## Как запустить проект

1. **Установить зависимости**
   ```bash
   cd sqlite_version
   pip install -r requirements.txt
   ```

2. **Запустить сервер**
   ```bash
   python app_backup.py
   ```

3. **Открыть браузер** и перейти: `http://127.0.0.1:5000`

4. **Остановить сервер**: нажать `Ctrl+C` в терминале

---

## Использование

1. Зарегистрируйтесь, введя никнейм и пароль
2. Войдите в чат под своим никнеймом
3. Напишите сообщение — оно появится у всех пользователей
4. Сообщения сохраняются в файле `chat.db` в папке проекта

---

## Отличия от MySQL-версии

| Особенность | SQLite | MySQL |
|-------------|--------|-------|
| База данных | Файл `chat.db` | Сервер MySQL |
| Поля пользователя | `nickname`, `password` | `name`, `surname`, `login`, `password` |
| Чат | Групповой | 1-на-1 |
| Сессии | `session["nickname"]` | `session["user_id"]`, `session["user_login"]` |
| Маршруты | `/login`, `/register`, `/chat`, `/send`, `/messages` | `/login`, `/register`, `/chat`, `/send_message`, `/get_messages` |
