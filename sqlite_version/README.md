# SQLite-версия ChatApp

> Простой и удобный чат прямо в браузере — без установки приложений. Версия на SQLite.

---

## Что это такое?

**ChatApp SQLite** — учебный веб-чат, где несколько пользователей могут переписываться в реальном времени.

---

## Возможности

- **Умное приветствие** — "Доброе утро", "Добрый день", "Добрый вечер" или "Доброй ночи" в зависимости от времени
- **Вход по логину и паролю** — авторизация через SQLite
- **Групповой чат** — все пользователи видят сообщения всех
- **История сообщений** — переписка сохраняется в базе данных SQLite
- **Время и автор** — у каждого сообщения видно кто и когда написал
- **Группировка по дням** — дата показывается один раз при переходе на новый день
- **Защита от чужого входа** — чужой логин без пароля не займёшь
- **Динамическое обновление** — сообщения появляются без перезагрузки страницы

---

## Технологии

| Что | Зачем |
|-----|-------|
| Python 3.12 + Flask | Сервер, маршруты, логика |
| SQLite | База данных (пользователи и сообщения) |
| HTML + Jinja2 | Страницы сайта |
| CSS | Внешний вид |
| JavaScript (fetch) | Динамика без перезагрузки |

## Структура проекта

```
sqlite_version/
│
├── app.py                ← главный файл: весь Python, маршруты, логика
├── chat.db               ← база данных SQLite (создаётся при первом запуске)

│
├── templates/            ← HTML-страницы (Flask ищет шаблоны здесь)
│   ├── index.html        ← страница входа
│   ├── registration.html ← страница регистрации
│   └── chat.html         ← страница чата (групповой)
│
└── static/               ← файлы, которые браузер скачивает напрямую

    └── chat.js           ← динамика без перезагрузки
```

## База данных SQLite

### Таблица users (пользователи)

| Поле | Тип | Описание |
|------|-----|----------|
| id | INTEGER (PK, AUTOINCREMENT) | Уникальный ID пользователя |
| name | TEXT | Имя пользователя |
| surname | TEXT | Фамилия пользователя |
| login | TEXT (UNIQUE, NOT NULL) | Логин |
| password | TEXT | Пароль |

### Таблица messages (сообщения)

| Поле | Тип | Описание |
|------|-----|----------|
| id | INTEGER (PK, AUTOINCREMENT) | Уникальный ID сообщения |
| sender_id | INTEGER | ID отправителя |
| text | TEXT | Текст сообщения |
| created_at | TEXT | Дата и время (YYYY-MM-DD HH:MM:SS) |

## Как запустить проект

1. **Установить зависимости** (если нужно)
   ```bash
   cd sqlite_version
   pip install flask
   ```

2. **Запустить сервер**
   ```bash
   python app.py
   ```

3. **Открыть браузер** и перейти: `http://localhost:5000`

4. **Остановить сервер**: нажать `Ctrl+C` в терминале

---

## Использование

1. Зарегистрируйтесь, введя логин и пароль
2. Войдите в чат под своим логином
3. Напишите сообщение — оно появится у всех пользователей (групповой чат)
4. Сообщения сохраняются в файле `chat.db` в папке `sqlite_version/`
