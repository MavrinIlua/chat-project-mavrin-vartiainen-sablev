# chat-project-mavrin-vartiainen-sablev

Пользователь открывает сайт:
Потом: index.html — вводит ник + пароль. 
Если норм: /login — Flask проверяет БД. 
В БД 3 варианта: 
1) Новый ник? 
2) Ник есть + пароль верный? 
3) Пароль неверный?

Если нет никнейма:
1) либо: Регистрируем 
2) либо: Авторизуем 
3) либо: Ошибка на index.html
Если норм: Сессия["nickname"] = "Илья (или другой nickname)"
Переход на страницу чата:
chat.html — страница чата
