# chat-project-mavrin-vartiainen-sablev

Пользователь открывает сайт
        ↓
    index.html — вводит ник + пароль
        ↓
    /login — Flask проверяет БД
        ↓
  Новый ник?          Ник есть + пароль верный?      Пароль неверный?
     ↓                        ↓                            ↓
Регистрируем         Авторизуем                   Ошибка на index.html
     ↓                        ↓
  Сессия["nickname"] = "Илья"
     ↓
  chat.html — страница чата

Вот логичное решение, исходя из того, что уже есть в проекте:
Задача из бэклога получаем за неделю:

| Что конкретно | Ответственный | Детали |
| :--- | :--- | :--- |
| Отправка сообщений (ID 3) | Илья | Форма отправки `index.html` + маршрут `/send` в `app.py` |
| Уникальные никнеймы (ID 4) | Илья | Это уже начато — форма входа в `index.html` есть |
| История в БД (ID 5) | Евгений | Подключить SQLite, создать таблицу, сохранение сообщений |
| Отображение времени и автора (ID 7) | Евгений | При сохранении в БД записи метки времени, выведите в чат. |
| Динамическая отправка (ID 6) | Анна | JavaScript на странице чата — отправка без перезагрузки |
