### 🗄️ **Структура базы данных MySQL**

```
Таблица users (пользователи):
  id        — автоинкрементный ID
  name      — имя пользователя
  surname   — фамилия пользователя
  login     — логин (уникальный)
  password  — пароль (хешированный)
  
Таблица messages (сообщения):
  id          — автоинкрементный ID
  sender_id   — ID отправителя (внешний ключ к users.id)
  receiver_id — ID получателя (внешний ключ к users.id)
  text        — текст сообщения
  created_at  — дата и время создания
```

### 🔌 **API endpoints**

| Метод | Endpoint | Описание |
|:---|:---|:---|
| POST | `/login` | Вход пользователя |
| POST | `/register` | Регистрация нового пользователя |
| GET | `/chat?user_id=X` | Страница чата с пользователем X |
| POST | `/send_message` | Отправка сообщения |
| GET | `/get_messages?user_id=X` | Получение истории переписки |

*Приоритеты: **Must** — обязательно, **Should** — очень желательно, **Could** — можно сделать, **Won't** — не в этот раз.*
